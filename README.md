# Crystalline Carbon 💎

**🔗 Live:** https://ada-555.github.io/crystalline-carbon/

---

A hyper-luxury visualization of a pure diamond lattice — blending forensic scientific clarity with high-jewelry elegance. Features complex refraction shaders that turn geometry into light.

## Features

- 🔮 **ATOMIC → DIAMOND Morph** — Scroll to crystallize chaotic particles into perfect lattice
- 💎 **True Diamond IOR** — Refraction index of 2.417 (actual diamond)
- 🌈 **Chromatic Dispersion** — Rainbow "fire" light splits via multi-channel refraction
- ⚛️ **FCC Lattice** — Face-centered cubic crystal structure
- 🔗 **Covalent Bonds** — Electrons flow along bond paths
- ✨ **High-Intensity Bloom** — Captures spectral fire
- 🔬 **Orbit Controls** — Extreme close-up analysis of lattice geometry
- 🌡️ **Dynamic Lighting** — Multiple spotlights for specular highlights

## Tech Stack

- Three.js (r160)
- MeshPhysicalMaterial with transmission
- Custom post-processing pipeline
- GitHub Pages
